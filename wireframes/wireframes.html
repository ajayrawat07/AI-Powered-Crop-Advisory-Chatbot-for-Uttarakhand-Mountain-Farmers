📋 Wireframe Specifications

Shared Components Across All Screens

  • Header: Logo, Navigation items, Theme toggle (dark/light)
  • Footer: Links, copyright, social media placeholders
  • Colors: Primary: #2d5016 (green), Secondary: #4a7c2c, Neutral: #666, #999, #ccc
  • Typography: Grey boxes for text placeholders (lo-fi style)

Screen Details

1. Home Screen: Hero section, CTA buttons, featured content cards

2. Dashboard: 4 main card components with data visualization placeholders

3. Detail/List View: Filterable list with expandable details

4. Login/Signup: Form with email, password, confirm password fields

5. AI Chat: Chat conversation area with message bubbles and input field

Responsive Breakpoints

  • Mobile: 375px (single column layout)
  • Tablet: 768px (2-column grid)
  • Desktop: 1440px (multi-column layout)

Additional Features

  • Dark/Light mode toggle in header
  • Hover states on buttons and interactive elements
  • Loading states with spinner/skeleton loaders
  • Toast notifications for feedback
  • Modal dialogs for confirmations
