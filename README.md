# AI-Powered Crop Advisory Chatbot for Uttarakhand Mountain Farmers

## One-Line Description
An AI-powered chatbot that provides crop advisory, disease diagnosis, pest management, and farming recommendations for farmers in Uttarakhand's mountain regions.

## Tech Stack
- Frontend: React.js
- Styling: Tailwind CSS
- Backend: FastAPI
- Database: MongoDB
- AI API: OpenAI API
- Deployment: Vercel + Render
